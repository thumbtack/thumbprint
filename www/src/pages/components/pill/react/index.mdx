## Pill variations

### Default

```jsx
<Pill>Available</Pill>
```

### Icon & color

```jsx
<Pill color="red" icon={<ContentModifierTrophyTiny />}>
    In high demand
</Pill>
```

### Multiple

```jsx
<div>
    <span className="mr2">
        <Pill color="blue">Highly rated</Pill>
    </span>
    <span>
        <Pill color="green" icon={<ContentModifierLightningTiny />}>
            Fast response
        </Pill>
    </span>

```
