## Usage

`AlertBanner` should be used as a banner at the very top of the page.

Call-to-actions (e.g. `<a>` tags in the `children` prop) should be the last item in the `children`.

## Themes

### Caution

```jsx
<AlertBanner theme="caution">
    Your pre-paid credits are running low. Purchase new credits before you run out.{' '}
    <a href="https://example.com/">Purchase credits</a>
</AlertBanner>
```

### Info

```jsx
<AlertBanner theme="info">
    You havenâ€™t finish setting up. Add 3 reviews and turn on targeting.{' '}
    <a href="https://example.com/">Finish setup</a>
</AlertBanner>
```

### Warning

```jsx
<AlertBanner theme="warning">
    Oops. Your payment failed! Update your payment details to continue using Thumbtack.{' '}
    <a href="https://example.com/">Update payment</a>
</AlertBanner>
```
