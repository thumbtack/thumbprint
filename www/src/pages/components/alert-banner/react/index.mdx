## Usage

AlertBanner should be used as a banner on top of the page, above the global header.

Call-to-actions (e.g. `<a>` tags in the `children` prop) should be the last item in the children.

## Themes

### Caution

```jsx
<AlertBanner theme="caution">
    This is a caution banner. <a href="#">Go to top of the page</a>
</AlertBanner>
```

### Info

```jsx
<AlertBanner theme="info">
    This is an info banner. <a href="#">Go to top of the page</a>
</AlertBanner>
```

### Warning

```jsx
<AlertBanner theme="warning">
    This is a warning banner. <a href="#">Go to top of the page</a>
</AlertBanner>
```
