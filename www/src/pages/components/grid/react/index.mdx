## Examples

### Basic grid

```jsx
<Grid>
    <GridColumn base={12} aboveSmall={4}>
        <div className="ba b-green h3" />
    </GridColumn>
    <GridColumn base={12} aboveSmall={4}>
        <div className="ba b-green h3" />
    </GridColumn>
    <GridColumn base={12} aboveSmall={4}>
        <div className="ba b-green h3" />
    </GridColumn>
</Grid>
```

### Wide gutters

```jsx
<Grid gutter="wide">
    <GridColumn base={4}>
        <div className="ba b-green h3" />
    </GridColumn>
    <GridColumn base={4}>
        <div className="ba b-green h3" />
    </GridColumn>
    <GridColumn base={4}>
        <div className="ba b-green h3" />
    </GridColumn>
</Grid>
```

### No gutters

```jsx
<Grid gutter="flush">
    <GridColumn base={4}>
        <div className="ba b-green h3" />
    </GridColumn>
    <GridColumn base={4}>
        <div className="ba b-green h3" />
    </GridColumn>
    <GridColumn base={4}>
        <div className="ba b-green h3" />

```

### Responsive at all breakpoints

```jsx
<Grid>
    <GridColumn base={12} aboveSmall={8} aboveMedium={6} aboveLarge={4}>
        <div className="ba b-green h3" />
    </GridColumn>
</Grid>
```
