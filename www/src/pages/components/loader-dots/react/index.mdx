## Loader Sizes

### Default

```jsx
<LoaderDots size="medium" />
```

### Small

```jsx

```

## Loader colors

This loader is available in multiple colors for use on elements with a non-white background.

### Brand

```jsx
<LoaderDots theme="brand" />
```

### Inverse

```jsx theme=dark
<LoaderDots theme="inverse" />
```

### Muted

```jsx
<LoaderDots theme="muted" />
```
