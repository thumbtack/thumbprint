## Loader Sizes

### Default

```html
<ul class="tp-loader">
    <li></li>
    <li></li>
    <li></li>
</ul>
```

### Small

```html
<ul class="tp-loader tp-loader--small">
    <li></li>
    <li></li>
    <li></li>
</ul>
```

## Loader colors

The loader can be used on dark backgrounds or in gray.

### Inverse

```html theme=dark
<ul class="tp-loader tp-loader--inverse">
    <li></li>
    <li></li>
    <li></li>
</ul>
```

### Muted

```html
<ul class="tp-loader tp-loader--muted">
    <li></li>
    <li></li>
    <li></li>
</ul>
```
