## Summary

Control for indicating a loading state inside a view, such as a network call on button tap.

![default](ios-loaderdots-brand-medium.gif)

## Public API

### `public var theme: Theme`

The following three themes are supported

`brand`

![brand](ios-loaderdots-brand-medium.gif)

`inverse`

![inverse](ios-loaderdots-inverse-medium.gif)

`muted`

![muted](ios-loaderdots-muted-medium.gif)

### `public var size: Size`

The following sizes are supported

`small` ![small](ios-loaderdots-brand-small.gif)

`medium` ![medium](ios-loaderdots-brand-medium.gif)
