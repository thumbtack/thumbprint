## Avatar Badges

<DeprecatedComponentAlert>
    Avatar badges are deprecated. Status indicators such as "online" status, hired checkmarks, and
    notification alerts should be implemented as separate elements placed next to the avatar.
</DeprecatedComponentAlert>

Badges can be added to avatars to denote information including hired status and unread notifications. These badges cannot be used on the `xsmall` variation.

### Checked

```jsx
<UserAvatar imageUrl="https://www.placecage.com/640/480" isChecked size="xlarge" />
```

### Notification

```jsx
<UserAvatar imageUrl="https://www.placecage.com/640/480" hasUnreadNotifications size="xlarge" />
```

### Online

```jsx
<UserAvatar imageUrl="https://www.placecage.com/640/480" isOnline size="medium" />
```
