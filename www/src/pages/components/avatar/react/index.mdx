## Avatar Sizes

### Extra Large

```jsx
<Avatar imageUrl="https://randomuser.me/api/portraits/women/74.jpg" size="xlarge" />
```

### Large

```jsx
<Avatar imageUrl="https://randomuser.me/api/portraits/men/36.jpg" size="large" />
```

### Medium

```jsx
<Avatar imageUrl="https://randomuser.me/api/portraits/men/54.jpg" size="medium" />
```

### Small

```jsx
<Avatar imageUrl="https://randomuser.me/api/portraits/women/58.jpg" size="small" />
```

### Extra Small

```jsx
<Avatar imageUrl="https://randomuser.me/api/portraits/women/84.jpg" size="xsmall" />
```

## Avatar Badges

Badges can be added to avatars to denote information including hired status and unread notifications. These badges cannot be used on the `xsmall` variation.

### Checked

```jsx
<Avatar imageUrl="https://randomuser.me/api/portraits/women/28.jpg" isChecked size="xlarge" />
```

### Notification

```jsx
<Avatar
    imageUrl="https://randomuser.me/api/portraits/women/47.jpg"
    hasUnreadNotifications
    size="xlarge"
/>
```

### Online

```jsx
<Avatar imageUrl="https://randomuser.me/api/portraits/men/51.jpg" isOnline size="medium" />
```
