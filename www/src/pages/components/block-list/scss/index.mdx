## Default

`tp-block-list__link` creates a link that takes up all of the space in the row.

```html
<ul class="tp-block-list">
    <li>Item One</li>
    <li>Item Two</li>
    <li><a href="#" class="tp-block-list__link">Item Three</a></li>
    <li>Item Four</li>
</ul>
```

## Flush

```html
<ul class="tp-block-list tp-block-list--flush">
    <li>Item One</li>
    <li>Item Two</li>
    <li>Item Three</li>
    <li>Item Four</li>
</ul>
```

## Bottom border

```html
<ul class="tp-block-list tp-block-list--all">
    <li>Item One</li>
    <li>Item Two</li>
    <li>Item Three</li>
    <li>Item Four</li>
</ul>
```

## Surrounding border

```html
<ul class="tp-block-list-group">
    <li>Item One</li>
    <li>Item Two</li>
    <li>Item Three</li>
    <li>Item Four</li>
</ul>
```
