## Basic Text Area example

```html
<label class="tp-label" for="example-basic-text-area">Business description</label>
<textarea
    class="tp-textarea"
    id="example-basic-text-area"
    placeholder="Tell us about your business"
/>
```

## Text Area with an error

The `tp-textarea--bad-news` class only changes the textarea's color. It should be used alongside an error message that helps users advance through the form.

```html
<textarea
    class="tp-textarea tp-textarea--bad-news"
    placeholder="Tell us about your business"
></textarea>
```
