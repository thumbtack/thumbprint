## Basic Switch

Switch can be checked or unchecked.

The `isChecked` prop determines if a switch is checked.

```jsx
function SwitchExample() {
    const [isChecked, setIsChecked] = React.useState(true);

    return <Switch isChecked={isChecked} name="example-basic" onChange={v => setIsChecked(v)} />;
}
```

## Large Switch

Switches are `small` by default. The `size` prop allows you to render a `large` switch.

```jsx
function SwitchExample() {
    const [isChecked, setIsChecked] = React.useState(true);

    return (
        <Switch
            isChecked={isChecked}
            size="large"
            name="example-basic"
            onChange={v => setIsChecked(v)}
        />
    );
}
```


## Disabled Switch

The `isDisabled` prop visually and functionally disabled switch.

```jsx
function SwitchExample() {
    const [isChecked, setIsChecked] = React.useState(true);

    return (
        <Switch
            isChecked={isChecked}
            isDisabled
            name="example-basic"
            onChange={v => setIsChecked(v)}
        />
    );
}
```
