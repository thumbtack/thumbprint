## Button themes

These are the available `Button` styles for use on light background colors.

```jsx
<div className="flex flex-column" style={{ gap: tokens.tpSpace3 }}>
    <ButtonRow>
        <PrimaryButton>Primary</PrimaryButton>
    </ButtonRow>

    <ButtonRow>
        <SecondaryButton emphasis="high">Secondary</SecondaryButton>
        <SecondaryButton emphasis="low">Secondary</SecondaryButton>
        <SecondaryButton isBordered emphasis="high">
            Secondary
        </SecondaryButton>
        <SecondaryButton isBordered emphasis="low">
            Secondary
        </SecondaryButton>
    </ButtonRow>

    <ButtonRow>
        <TertiaryButton>Tertiary</TertiaryButton>
    </ButtonRow>

    <ButtonRow>
        <CautionButton emphasis="high">Caution</CautionButton>
        <CautionButton emphasis="low">Caution</CautionButton>
    </ButtonRow>
</div>
```

## Icon within a button

The `iconLeft` and `ironRight` props accept any small icon from [Thumbprint Icons](/icons).

```jsx
<div className="flex flex-column" style={{ gap: tokens.tpSpace3 }}>
    <ButtonRow>
        <PrimaryButton iconLeft={<ContentModifierMessageSmall />}>Send Message</PrimaryButton>
    </ButtonRow>
    <ButtonRow>
        <SecondaryButton iconRight={<ContentModifierMessageSmall />} isBordered>
            Send Message
        </SecondaryButton>
    </ButtonRow>
</div>
```
