## Banner alerts

These alerts should be used at the very top of a page.

```jsx
<BannerAlert messageType="bad">
    <p>An Alert!</p>
</BannerAlert>
```

```jsx
<BannerAlert messageType="good">
    <p>An Alert!</p>
</BannerAlert>
```

```jsx
<BannerAlert messageType="warning">
    <p>An Alert!</p>
</BannerAlert>
```

```jsx
<BannerAlert messageType="note">
    <p>An Alert!</p>
</BannerAlert>
```

## In-page alerts

These alerts should be used inside other page elements, e.g. forms.

```jsx
<InPageAlert messageType="bad">
    <p>
        Your credit card is about to expire. To keep quoting on job requests please
        <a href="https://example.com/">update your credit card now</a>.
    </p>
</InPageAlert>
```

