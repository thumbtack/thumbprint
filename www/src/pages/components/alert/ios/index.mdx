## Summary

Alerts on iOS are implemented using the built-in UIKit component `UIAlertController` with `preferredStyle: .alert`.

![iOS Alert](/img/ios-alert.png)

## Public API

See Apple documentation on [UIAlertController](https://developer.apple.com/documentation/uikit/uialertcontroller) for the public API.
