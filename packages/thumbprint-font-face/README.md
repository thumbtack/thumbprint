# Thumbprint Font Face

Provides the `@font-face` CSS needed to render Thumbtack fonts.

## SCSS usage

```scss
$thumbprint-font-url: '[SERVER URL HERE]';
@import '@thumbtack/thumbprint-font-face/_index';
```
